# vuejs에 폰트어썸 사용

화면 개발할때 아이콘을 자주 사용합니다. 아이콘을 사용하면 사용자가 해당 기능에 대한 이해가 수월해 지고 만든 화면의 완성도도 높아진다는 장점이 있습니다. 하지만 아이콘이 필요할때 마다 디자이너에게 요청하면 상당히 눈치가 보일것 같네요. 그렇다고 포토샵을 열어서 아이콘을 만들수도 없고.. (20대 때는 제가 포토샵 으로 만들어서 사용했던것 같아요) 이때 사용할수 있는 유용한 웹 아이콘 폰트 라이브러리가 있습니다. Font Awesome 이라는 라이브러리 인데요 웹 아이콘 폰트 시장에서 가장 인기있는 라이브러리 입니다. 이번 시간에는 Vue.js 에서 Font Awesome 을 사용하는 방법에 대해서 알아 보겠습니다.

# 1. 설치

Vue.js 프로젝트를 생성했다고 가정하고 font awesome 을 추가 설치 하는 방법입니다. yarn 또는 npm 둘중 하나의 방법을 이용해 라이브러리를 추가해 주세요

# yarn을 이용한 설치

yarn 을 이용한 설치 방법입니다.

yarn add @fortawesome/fontawesome-free
1

# npm 설치 방법

npm을 이용한 설치 방법 입니다.

npm install --save @fortawesome/fontawesome-free
1

# main.js 수정

main.js 에 아래 코드를 추가합니다.

import '@fortawesome/fontawesome-free/js/all.js'
1

# 2. 사용법

폰트어썸 사용하는 방법입니다.

vue.js template 부분에 아래와 같이 코드를 작성하시면 아이콘이 출력됩니다.

<i class="fa fa-bus"></i>
<i class="fas fa-ambulance"></i>
<i class="fas fa-hotel"></i>

2

# 3. Icon 타입

https://fontawesome.com/cheatsheet (opens new window) 에 접속하시면 무료로 사용할수 있는 아이콘들을 확인할수 있습니다. 여기서 제공해준 아이콘 리스트를 참고하여 아이콘 검색화면과 아이콘 색인 화면을 만들어 보았습니다. 검색 또는 색인 결과 아이콘을 클릭하시면 클립보드에 해당 내용이 복사 되며 vue 프로젝트에서 붙여 넣기 해서 사용하시면 됩니다.

# 4. 색상변경

폰트어썸은 이미지가 아닌 svg vector 를 이용하여 만들었습니다. 이렇게 벡터를 만들어서 사용하면 font-size 를 크게 하거나 색상을 임의로 변경할수 있습니다. 아래 예제를 참고하세요

# 예제

  • 아이콘을 출력해 봅니다.
  • 아이콘의 색상을 변경해 봅니다.
  • 아이콘의 크기를 변경해 봅니다.
  • 아이콘의 테두리를 적용합니다.
  • 아이콘의 방향을 변경합니다.
  • 아이콘을 회전합니다.
  • 아이콘을 중첩합니다. 
<template>
  <div>
    <ul>
      <li>
        아이콘을 출력해 봅니다. <i class="fa fa-bus"></i>
      </li>
      <li>
        아이콘의 색상을 변경해 봅니다. <i class="fa fa-bus" style="color:blue"></i>
      </li>
      <li>
        아이콘의 크기를 변경해 봅니다.
        <i class="fa fa-bus"></i>
        <i class="fa fa-bus fa-lg"></i>
        <i class="fa fa-bus fa-2x"></i>
        <i class="fa fa-bus fa-3x"></i>
        <i class="fa fa-bus fa-4x"></i>
        <i class="fa fa-bus fa-5x"></i>
        <i class="fa fa-bus fa-6x"></i>
      </li>
      <li>
        아이콘의 테두리를 적용합니다. <i class="fa fa-bus fa-border"></i>
      </li>
      <li>
        아이콘의 방향을 변경합니다.
        <i class="fa fa-bus"></i>
        <i class="fa fa-bus fa-rotate-90"></i>
        <i class="fa fa-bus fa-rotate-90"></i>
        <i class="fa fa-bus fa-rotate-180"></i>
        <i class="fa fa-bus fa-rotate-270"></i>
        <i class="fa fa-bus fa-flip-horizontal"></i>
        <i class="fa fa-bus fa-flip-vertical"></i>
      </li>
      <li>
        아이콘을 회전합니다.
        <i class="fa fa-bus fa-spin fa-lg"></i>
      </li>

        아이콘을 중첩합니다.

        <i class="fa fa-circle fa-stack-2x"></i>
        <i class="fa fa-bus fa-stack-1x fa-inverse"></i>
        </span>
      </li>
    </ul>
  </div>
</template>
