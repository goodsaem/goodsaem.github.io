# 8. aws freeiter linux 서버에 openjdk 1.8 설치하기

# 1. 설치

백엔드 서비스는 springboot에서 모두 처리할 예정입니다. springboot을 기동하기 위해 open jdk를 설치합니다.

ubuntu@goodsaem:~$ sudo apt install openjdk-8-jdk
