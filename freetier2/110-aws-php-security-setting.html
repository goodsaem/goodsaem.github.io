# 10. AWS 보안설정

http 및 https 로 접속하기 위해서는 aws에 인바운드 (들어오는쪽)에 방화벽 설정을 해주셔야 됩니다. 아래와 같이 aws 콘솔에서 EC2 > 인스턴스 > 인스터스 명을 클릭하시고 보안 탭에서 보안그룹의 보안 그룹번호를 클릭합니다.

인바운드 규칙 변경을 통해 아래와 같이 http , https 에대한 규칙을 추가합니다.

규칙 추가는 인바운드 규칙 편집 버튼을 클릭하여 아래와 같이 http,https 위치무관이라고 선택하고 저장하시면 됩니다.
