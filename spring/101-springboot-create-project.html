# 1. SpringBoot 프로젝트 생성

spring boot 를 이용하여 restapi 서버를 구축 하고자 합니다. 관련된 내용을 정리했습니다.

# 1. 개발 툴 다운로드

Intellij IDEA (opens new window) 역시 java 개발 툴 끝판왕을 다운로드 받으시고 pc에 설치를 진행해 주세요.

# 2. spring project 생성

spring starter (opens new window) 사이트에서 spring 프로젝트를 생성합니다. 아래와 같이 설정을하고 GENERATE 버튼을 클릭하면 해당 설정되로 압축 파일을 생성해서 다운로드가 진행됩니다. 다운로드후 압축 파일 풀어서 적당한 위치에 옮겨 주세요 해당 디렉토리가 프로젝트 폴더가 됩니다.

구분 설정 비고
Project Gradle Project 를 선택합니다.
Language Java 를 선택합니다.
Spring Boot 2.4.2 를 선택합니다.
Group io.github.goodsaem GroupId는 프로젝트를 식별해 주는 고유값입니다.
각자의 도메인으로 설정해 주세요
Artifact api war 나 jar 파일을 만들때 사용되면 버전정보를 제외한 파일이름을 뜻합니다.
Name api
Description rest api
Package name io.github.goodsaem.api 각자 맞개 설정해 주세요
Packaging jar
Java 11 너무 높은 버전은 문제가 될수 있습니다. 안정버전을 선택해 주세요
Dependencies Lombok getter 나 setter 등을 자동완성할때 사용합니다.
H2 Database H2 db 사용을 위함 입니다.
MySQL Driver mysql db 연결을 위한 드라이버를 추가합니다.
Spring JPA Java Persistence API 로 객체 지향적으로 데이터를 관리 할수 있습니다.

Intellij 를 실행하고 open 버튼을 클릭합니다.

WARNING

jdk 가 설치되어 있지 않다면 다음 단계 진행이 되지 않습니다. oracle jdk 1.8.20x 버전부터 유료화 되었으므로 openjdk 를 설치하여 사용하시기 바랍니다. openjdk download (opens new window)

압축푼 api 폴더를 아래와 같이 선택한후 open 버튼을 클릭해 주세요.

install 이 자동으로 진행됩니다.

프로젝트 생성이 끝나고 나면 좌측에 build.gradle 파일을 열어서 아래 강조한 부분을 추가해 주세요 freemarker 는 java template 문법을 사용하기 위해서 사용되고 spring starter web 은 내장된 was (web application server) 를 사용하기 위해 추가 합니다. Gradle Tab 의 sync 아이콘을 클릭하면 추가한 패키지를 다운로드 받아서 프로젝트 자동으로 추가 됩니다. 참 편하죠?























 
 











plugins {
	id 'org.springframework.boot' version '2.4.2'
	id 'io.spring.dependency-management' version '1.0.11.RELEASE'
	id 'java'
}

group = 'io.github.goodsaem'
version = '0.0.1-SNAPSHOT'
sourceCompatibility = '11'

configurations {
	compileOnly {
		extendsFrom annotationProcessor
	}
}

repositories {
	mavenCentral()
}

dependencies {
	implementation 'org.springframework.boot:spring-boot-starter-data-jpa'
	implementation 'org.springframework.boot:spring-boot-starter-freemarker'
	implementation 'org.springframework.boot:spring-boot-starter-web'
	compileOnly 'org.projectlombok:lombok'
	runtimeOnly 'com.h2database:h2'
	runtimeOnly 'mysql:mysql-connector-java'
	annotationProcessor 'org.projectlombok:lombok'
	testImplementation 'org.springframework.boot:spring-boot-starter-test'
}

test {
	useJUnitPlatform()
}

intellij 에서 Lombok 플러그인을 사용하기 위해 Preference > Plugins - Lombok 을 검색하여 install 한후 Intellij 를 재시작 합니다.

Preference > Build, Execution,Deployment > Compiler > Annotation Processors 선택 하고 오른쪽 화면의 Enable annotation processing 을 체크한후 OK 버튼을 클릭합니다. 체크해야만 Lombok 플러그인을 사용할수 있습니다.
