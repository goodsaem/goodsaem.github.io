# 1. 변수와 상수

변수는 값이 계속 바뀔때 사용하고 상수는 한번 선언하고 나서 값이 바뀌지 않는 경우에 사용합니다. 보다 자세한 내용에 대해서 알아보겠습니다.

# 1. 변수

# (1) let

1 부터 n 까지 더하는 프로그램을 만들어 보겠습니다. 제가 요즘 vue.js 를 공부하고 있어 vue.js 로 표현하는 예제가 많을텐데요 vue.js는 따로 정리를 할테니 우선 javascript 부분에만 집중해서 보시면 되겠네요. start 와 end 값을 바꾸어 보면 sum 값이 계속 변경되는걸 알수 있습니다.

start : end : sum : 1 + 10 = 55

아래 검은색으로 강조한 부분 코드보시면 let sum 이라는 변수를 선언한후 start end 까지 반복문을 실행하면서 sum 변수에 값을 변화시키고 있습니다. 이렇듯 변하는 값을 넣을때 let 을 사용하셔야 되구요 let 으로 선언되면 변수입니다.





















 
 
 
 














<template>
  <div class="exam1">
    start : <input v-model="start" />
    end : <input v-model="end" />
    sum : {{ start }} + {{ end }} = {{ sumNumber }}
  </div>
</template>

<script>
export default {
  name: "Exam1",
  data() {
    return {
      start : 1,
      end : 10,
      sum : 0
    }
  },
  computed: {
    sumNumber() {
      let sum = 0;
      for(let i=parseInt(this.start); i<=parseInt(this.end); i++) {
        sum += i;
      }
      this.sum = sum;
      return this.sum;
    }
  }
}
</script>

.exam1 {
  background-color: lightblue;
  padding: 8px;
}

# (2) 전역변수와 지역변수

변수에는 전역변수와 지역 변수가 있습니다. 우선 예제를 보시죠.

  • 실행결과

    • test1() : 20
    • test2() : 30
    • test3() : 30
    let global_a 해서 전역 변수를 선언했습니다. 아래 그림을 보시면 알수 있듯이 global_a 는 녹색 영역 및 노란색 영역 모두다 사용할수 있는 변수입니다. 전체 영역에서 사용할수 있다고 해서 전역변수라고 합니다. local_b 변수는 노란색 박스 영역에서만 사용할수 있습니다. 노란색 박스 영역을 넘어서면 사용할수 없습니다. 그래서 지역내에서만 사용할수 있다고 하여 지역변수 입니다.

  • 소스코드 global_a 변수에는 10이란 값이 있었습니다. test1 함수에서는 global_a 10 과 local_b 값 10이 더해져서 20값을 리턴했습니다. 다시 test2 함수 에서 global_a 변수값을 20으로 바꾸고 로컬변수 local_b = 20 과 합해져서 30값을 리턴했습니다. 다시 test1 함수를 호출하면 test2 함수에서 global_a 값울 변경하여 30으로 리턴되었습니다.












 
 
 
 
 
 
 
 
 
 
 
 
 
 
 

























<template>
  <div class="exam">
    <ul>
      <li>test1() : {{ test1 }}</li>
      <li>test2() : {{ test2 }}</li>
      <li>test3() : {{ test3 }}</li>
    </ul>
  </div>
</template>

<script>
let global_a = 10;
function test1() {
  let local_b = 10;
  return global_a + local_b;
}

function test2() {
  global_a = 20;
  let local_b = 10;
  return global_a + local_b;
}

function test3() {
  return test1();
}

export default {
  name: "Exam2",
  created() {
    this.test1 = test1();
    this.test2 = test2();
    this.test3 = test3();
  },
  data() {
    return {
      test1  : 0,
      test2  : 0,
      test3  : 0,
    }
  },
}
</script>

<style scoped>
.exam {
  background-color: lightblue;
  padding: 8px;
}
</style>

# (3) 상수

const 로 선언 되면 상수입니다. 한번 선언되고 나서 값이 바뀌지 않는 경우에는 const 를 사용하여 상수로 선언합니다.

const test = 1;
test = 2;
1
2

webstorm에서 위에 코드를 실행해보면 아래와 같이 오류가 발생합니다. const 는 readonly 변수 이므로 값을 할당할수 없다는 오류 메시지가 발생합니다. 이런 코드를 작성해서 크롬개발자 도구에서 console 창을 확인하면 오류가 발생합니다. const 는 1회성으로 변하지 않는 값을 할당 할때만 사용해 주세요.

원의 둘레는 구하는 공식은 2πr 입니다. 이때 파이를 약 3.14라고 하면 이값은 변하지 않는값 즉 상수 이고 반지름은 r 은 변하는 변수로 선언합니다. 이런 내용으로 간단한 예제를 만들어 보겠습니다. vue.js와 vue-konva 를 이용하여 원의 둘레를 구하는 예제를 만들어 보겠습니다. 아래 text 박스에서 반지름 값을 변경하면 원의 크기가 변경되고 원둘레 값도 실시간으로 변경됨을 확인할수 있습니다.

아래 83-86라인 선명하게 표시되는 부분의 코드를 보면 pi 값을 const 상수로 선언하고 반지름 r 과 둘레 l 은 변수 let 으로 선언하여 원의 둘레를 구하고 있습니다.

vue.js

vue.js는 자바스크립트 부분 작성이 끝나고 나면 정리해서 올리도록 하겠습니다. 조금만 기다려 주세요 지금은 javascript 코드에만 집중하시면 되겠네요.



















































































 
 
 
 
















<template>
  <div class="exam">
    <ul>
      <li>
        π : <input type="text" v-model="3.14" disabled> 
        반지름 : <input type="text" v-model="radius"></li>
    </ul>
    <v-stage :config="configKonva" ref="stage">
      <v-layer>
        <v-line :config="configLine"></v-line>
        <v-line :config="configLine2"></v-line>
        <v-circle :config="configCircle"></v-circle>
        <v-text :config="configText1"></v-text>
        <v-text :config="configText2"></v-text>
        <v-text :config="configText3"></v-text>
      </v-layer>
    </v-stage>
  </div>
</template>

<script>
export default {
  name: "Exam3",
  created() {
    this.radius = this.configCircle.radius;
  },
  data() {
    return {
      radius: 100,
      configKonva: {
        width: 500,
        height: 500,
      },
      configLine: {
        points: [0,250,500,250],
        stroke: 'green',
        strokeWidth: 2,
        dash: [6, 4],
      },
      configLine2: {
        points: [250,0,250,500],
        stroke: 'green',
        strokeWidth: 2,
        dash: [6, 4],
      },
      configCircle: {
        x: 500 / 2,
        y: 500 / 2,
        radius: 100,
        stroke: 'black',
        strokeWidth: 1,
      },
      configText1: {
        x: 10,
        y: 10,
        text: '',
        fontSize: 15,
        fontFamily: 'Iropke Batang',
        fill: 'black',
      },
      configText2: {
        x: 10,
        y: 30,
        text: '',
        fontSize: 15,
        fontFamily: 'Iropke Batang',
        fill: 'black',
      },
      configText3: {
        x: 10,
        y: 50,
        text: '',
        fontSize: 15,
        fontFamily: 'Iropke Batang',
        fill: 'black',
      },
    }
  },
  watch: {
    radius: {
      immediate: true,
      handler(newVal) {
        const pi = 3.14;
        this.configCircle.radius = newVal;
        let r = this.configCircle.radius;
        let l = 2 * pi * r;
        this.configText1.text = 'π : ' + pi;
        this.configText2.text = '반지름 : ' + r;
        this.configText3.text = '원둘레 : ' + l;
      }
    }
  }
}
</script>

}

# (4) 데이터 타입

변수와 상수에 넣은수 있는 값은 여러가지가 있습니다. 이 중 대표적인 것들에 대해서 알아 보겠습니다.

구분 예제 비고
숫자 let value = 1 계산용으로 사용됩니다.
문자열 let name = '좋은선생님' 작은 따옴표나 큰따옴표 안에 넣어주면 문자열 입니다.
참/거짓 let isGood = true
let isGood = false		 참 또는 거짓의 값을 넣을때 사용합니다.
null let girlfriend = null 값이 없음을 의미합니다.
undefined let girlfriend
console.log(girlfriend)		 undefined 값이 출력됩니다. 즉 아직값이 정의되지 않았다는 거죠
