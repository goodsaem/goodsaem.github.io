# 2. 연산자

연산자는 특정 연산을 수행하는 문자 입니다. 기초적인 부분이지만 간단히 정리 해 보겠습니다.

# 1. 대입 연산자

= 는 대입 연산자 입니다. 우측에 있는 값을 좌측에 입력 하는 역할을 담당 합니다. 변수 및 상수 에서 많이 보셨죠? value1 에 0을 대입하고 value2 에도 0을 대입하고 value3 에는 value1 + value2 값을 대입 하는 예제 입니다.

그리고 let value4+= value3 이런 씩으로 도 사용 하기도 하는데 이건 let value4 = value4 + value3 계산한 결과와 동일합니다. 즉 value4가 가지고 있는 값 value3 값을 합쳐서 value4에 입력하겠다는 뜻입니다. value5 -= value3 기존 가지고 있는 값에 value3을 뺀 결과를 대입 한다는 의미 입니다.

아래 코드의 강조 되는 부분만 보시면 실행 결과를 예측할수 있습니다.

  • let value1 =
  • let value2 =
  • let value3 = 0 + 0
  • value3 의 값은 0 입니다.
  • value4 += 0
  • value4 의 값은 0 입니다.
  • value5 -= 0
  • value5 의 값은 0 입니다.
















































 



 



 













<template>
  <div class="exam1">
    <ul>
      <li>
        let value1 =   <input type="number" v-model="value1" />
      </li>
      <li>
        let value2 =   <input type="number" v-model="value2" />
      </li>
      <li>
        let value3 =   {{ value1 }} + {{ value2 }}
      </li>
      <li>
        value3 의 값은  <b>{{ val3 }}</b> 입니다.
      </li>
      <li>
        value4 += {{ val3 }}
      </li>
      <li>
        value4 의 값은  <b>{{ val4 }}</b> 입니다.
      </li>
      <li>
        value5 -= {{ val3 }}
      </li>
      <li>
        value5 의 값은  <b>{{ val5 }}</b> 입니다.
      </li>
    </ul>

  </div>
</template>

<script>
export default {
  name: "Exam1",
  data() {
    return {
      value1 : 0,
      value2 : 0,
      value3 : 0,
      value4 : 0,
      value5 : 0,
    }
  },
  computed: {
    val3() {
      this.value1 = this.value1 === '' ? 0 : parseInt(this.value1);
      this.value2 = this.value2 === '' ? 0 : parseInt(this.value2);
      this.value3 =  this.value1 + this.value2;
      return this.value3;
    },
    val4() {
      this.value4 +=  parseInt(this.value3);
      return this.value4;
    },
    val5() {
      this.value5 -=  parseInt(this.value3);
      return this.value5;
    }
  }
}
</script>

# 2. 산술 연산자

산술 연산자 는 더하기 빼기 나누기 곱하기 나머지 등의 연산을 수행 하는 연산자 입니다.

  • + : 덧셈
  • - : 뺄셈
  • × : 곱셈
  • / : 나눗셈
  • % : 나머지
  • ++ : 증가 연산자
    • 자기가 가지고 있는 값을 1증가 하고 싶을때 사용 하는 연산자 입니다. 이 연산자 는 변수 앞에 붙을 때와 변수 뒤에 붙을때 결과가 다르므로 유의해서 사용하시기 바랍니다.
  • -- : 감소 연산자
    • 1감소할때 사용하는 연산자 입니다. 이부분도 앞/뒤 붙는 위치에 따라 결과가 달라 집니다.

산술 연산자가 나오니 계산기를 만들고 싶네요 간단히 만들어 보겠습니다.

숫자 버튼을 먼저 클릭해 주세요 그러면 좌측에 값이 들어 갑니다.. 그리고 나서 연산자를 누르세요 그리고 나서 숫자를 클릭하시면 우측에 값이 들어갑니다. 그상태에서 = 버튼을 클릭하면 결과가 나오는 난해한 계산기 입니다. 사칙연산이 어떻게 동작하는지 확인하시면 되구요

그리고 그다음 예제는 증감 연산자가 동작하는 모습을 보여드리고 싶어 예제를 만들었습니다. ++a a++ 버튼을 클릭해보면서 a와 결과값이 어떻게 변하는지 확인해 보세요.

  • 사칙연산 결과 :
  • 계산과정 :
  • ----------------------------------------------------------
    증감결과 :
    a 값 : 1 
<template>
  <div class="exam2">
    <ul>
      <li>
        사칙연산 결과  : <input type="text" disabled="true" v-model="result" />
      </li>
      <li>
        계산과정  :
        <input type="text" disabled="true" v-model="num1" />
        <input type="text" disabled="true" size="3" v-model="operator" />
        <input type="text" disabled="true" v-model="num2" />
      </li>
      <li>
        <input type="button" v-for="i in 10" :value="i-1" @click="numClick(i-1)">
      </li>
      <li>
        <input type="button" value="+" @click="operClick('+')"/>
        <input type="button" value="-" @click="operClick('-')"/>
        <input type="button" value="*" @click="operClick('*')"/>
        <input type="button" value="/" @click="operClick('/')"/>
        <input type="button" value="%" @click="operClick('%')"/>
      </li>
      <li>
        <input type="button" value="=" @click="calc" />
      </li>
      <li>
        ----------------------------------------------------------<br/>
        증감결과  : <input type="text" disabled="true" v-model="r" /> <br/>
        a 값 : {{ a }}
      </li>
      <li>
        <input type="button" value="++a" @click="calc2(1)" />
      </li>
      <li>
        <input type="button" value="a++" @click="calc2(2)" />
      </li>
      <li>
        <input type="button" value="--a" @click="calc2(3)" />
      </li>
      <li>
        <input type="button" value="a--" @click="calc2(4)" />
      </li>
    </ul>

  </div>
</template>

<script>
export default {
  name: "Exam2",
  data() {
    return {
      num1 : '',
      num2 : '',
      operator : '',
      result : 0,
      a : 1,
      r : 1,
    }
  },
  methods: {
    numClick(num) {
      if(this.num2 === '' && this.operator === '') {
        this.num1 += num;
      } else {
        this.num2 += num;
      }
    },
    operClick(oper) {
      this.operator = oper;
    },
    calc() {
      if(this.num1 !== '' && this.num2 !== '' && this.operator != '') {
        if(this.operator === '+') {
          this.result =  parseInt(this.num1) + parseInt(this.num2)
        } else if(this.operator === '-') {
          this.result = parseInt(this.num1) - parseInt(this.num2)
        } if(this.operator === '*') {
          this.result =  parseInt(this.num1) * parseInt(this.num2)
        } if(this.operator === '/') {
          this.result =  parseInt(this.num1) / parseInt(this.num2)
        } if(this.operator === '%') {
          this.result =  parseInt(this.num1) % parseInt(this.num2)
        }
        this.num1 = '';
        this.num2 = '';
        this.operator = '';
      } else {
        this.result =  0;
      }
    },
    calc2(c) {
      if(c === 1) {
        this.r = ++this.a;
      } else if(c === 2) {
        this.r = this.a++;
      } else if(c === 3) {
        this.r = --this.a;
      } else if(c === 4) {
        this.r = this.a++;
      }
    }
  }
}
</script>

# 3. 논리 연산자

true 또는 false를 값을 받았을때 처리하는 연산자 입니다.

  1. ! : not 연산을 수행합니다.
  2. && : and 연산을 수행합니다.
  3. || : or 연산을 수행합니다.

# Not 연산자

true 를 false 로 , false를 true로 변경합니다.

주로 어떤 연산의 결과가 true가 아닌경우 즉 false 인경우 !으로 바꿔서 true로 변환해서 값을 처리하는데 사용합니다. 바로 예제를 보시죠

  • 1. 숫자
  • 2. 위에 숫자의 +,- 버튼을 클릭하여 숫자를 증가또는 감소하세요
  • 3. 조건 : 숫자가 0보다 크면 true 작으면 false
  • 4. Not 스위치 : 오른쪽으로 옮기면 not 연산을 수행합니다.
  • 5.결과 : true

위 예제는 숫자가 0보가 크면 true 작으면 false 를 리턴하는 간단한 예제 인데로 ! 연산자를 사용하여 true일때는 false false일때는 true 로 변경하여 결과값을 변경했습니다. 아래 코드중 44번 라인만 참고를 하시면 되겠네요..

# && (and) || (or) 연산자

&& 연산자는 두조건 모두 참일때 true를 리턴하고 or 연산자는 둘중 하나만 참이라도 true 를 리턴합니다.

&& || 연산자 결과

true true true ture
true false false true
false false false false

논리 연산을 이용한 간단한 로그인을 구현해 보겠습니다.

  • 아이디 : admin 암호 : passwd
  1. 아이디와 패스워드를 입력받는 화면이 나옵니다. 해당화면에는 취소와 확인 버튼이 있습니다.
  2. 아이디를 입력 받습니다.
  3. 취소를 누르면 "로그인을 취소하셨습니다." 라고 하고 로그인과정을 중단합니다.
  4. 확인 버튼을 클릭합니다.
  5. 입력받은 아이디가 admin 이 아니라면 "아이디가 등록되지 않았습니다." 라고 결과를 출력합니다.
  6. 입력받은 아이디가 admin 이고 패스워드가 passwd 가 아니면 "비밀번호가 틀렸습니다." 라고 결과를 출력합니다.
  7. 아이디 패스워드가 모두 동일하면 "안녕하세요 admin 님" 이라고 출력합니다.
