# 안녕하세요

안녕하세요 좋은 선생님 goodsaem 입니다. 최근 vue.js 로 개발할 일이 생겼습니다. x-platform , jquery 이런 걸로 개발해 오다가 vue.js는 정말 달랐습니다. 프로그래머의 생명 연장 프로젝트의 일환으로 javascript , vue.js , spring , java, sql 이런 내용들을 정리할려고 합니다. 매번 같은 내용을 구글링 해서 찾아 다니는것도 힘들어서 못하겠네요 최대한 자세하게 예제도 많이 남겨서 바로 사용할수 있는 수준까지 정리해 보겠습니다. 이 다짐이 변치 않고 지속적으로 이어 갈수 있기를 기대 하며 블로그를 찾아와 주신 분에게 고마운 마음을 전달합니다.

# 이 블로그는 어떻게 만들었나요 ?

기술 블로그는 내용을 많은 내용을 볼수 있으면 좋겠다는 생각을 했습니다. 그런 제 취양에 맞는 블로그 툴을 찾아다니다가 vuepress 라는 정적 생성기가 있어 vue 도 사용하고 블로그도 되면 좋겠다는 생각이 들어 vuepress 를 사용하게 하게 되었습니다. vuepress 이전에는 jekyll 이나 hexo 같은 것들로 블로그를 만들었는데 확실히 vuepress 가 좋군요. 블로그에 vue 로 만들어진 컴포넌트들을 자유롭게 가져다가 쓸수 있는 부분이 가장 큰 장점 이였습니다. 그리고 블로그에서 예제를 바로바로 실행해 보고 싶은데 다른 블로그 툴들은 제약이 있습니다. vuepress 는 바로바로 쉽게 vue component 를 실행 할수 있는 장점이 있습니다. 시간이 되면 이 블로그를 어떻게 만들었는지는 따로 정리 하겠습니다.

# 블로그에 사용한 도구들

  • vuepress (opens new window) 를 사용했습니다.
  • 블로그에 자유롭게 그림을 그리기 위해 canvas 기반의 드로잉 라이브러리 vue-konva (opens new window) 를 사용했습니다.
  • 클래스 다이어그램 작성을 위해 markdown-it-plantuml (opens new window) 을 사용했습니다.
  • 아이콘 사용을 위해 fontawesome (opens new window) 을 사용했습니다.
  • combobox 나 datepicker 같은걸 사용하기 위해 element (opens new window) 를 사용했습니다.
  • grid 는 tui-grid (opens new window) 를 사용했습니다.
  • 블로그에 댓글을 사용하기 위해 github issue 와 연동해서 사용하는 utterances (opens new window) 를 사용했습니다.

# 첫번째 코드

Hello goodsaem! 
<template>
  <div>
    <input type="text" v-model="hello">
    {{ hello }}
  </div>
</template>

<script>
export default {
  name: "HelloGoodSaem",
  data() {
    return {
      hello : 'Hello goodsaem!'
    }
  }
}
</script>
